Parameters

inside_lcfs¶

A 2D function that identifies if a given (r, z) coordinate lies inside or outside the plasma Last Closed Flux Surface (LCFS). This mask function returns a value of 1 if the requested point lies inside the LCFS. A value of 0.0 is returned outside the LCFS.

inside_limiter¶

A 2D function that identifies if a given (r, z) coordinate lies inside or outside the first wall limiter polygon. This mask function returns a value of 1 if the requested point lies inside the limit polygon. A value of 0.0 is returned outside the polygon.
